 ## Blumaps Dev Setup

 ### Clone Repo

 ```bash
 git clone <repo>

 ```


  ```bash
 cd blumap

 ```

  ```bash
 pip install requirements

 ```

  ```bash
 python makemigrations migrate

 ```

  ```bash
 python manage.py migrate

 ```
### Run on local dev (http://localhost:8000)
  ```bash
 python manage.py runserver

 ```

 ### Run tests
  ```bash
 python manage.py tests

 ```
